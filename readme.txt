# 850and1linux 🐧

A curated list of 851 Linux distributions. Because someone had to.

---

## What is this?

A single `.txt` file with **851 Linux distros**, one per line, sorted alphabetically. From the mainstream giants like **Ubuntu**, **Fedora** and **Arch**, all the way down to the obscure, forgotten, and questionable corners of the Linux world.

## Why 851?

Why not.

## Usage

```bash
# Clone it
git clone https://github.com/yourusername/850and1linux.git

# Count them yourself if you don't trust me
wc -l Distribuciones_Linux_851.txt

# Pick a random distro to install at 2am
shuf -n 1 Distribuciones_Linux_851.txt
```
